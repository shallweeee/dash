Dash 설치
```
conda create -y -n dash python=3.8.5
conda activate dash
pip install -r requirements.txt
```

Dash 실행

- 터미널
```
(dash) $ python app.py
```
- VSCODE
```
F5
또는
Ctrl + Shft + D > Python: Dash > Start Debugging
```


[VSCODE 단축키](https://demun.github.io/vscode-tutorial/shortcuts/)
